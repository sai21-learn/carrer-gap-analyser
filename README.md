# CareerCompass AI (Phase 2: Frontend Migration)

## Overview
This branch implements the modern web frontend for CareerCompass AI, replacing the Streamlit dashboard with a robust Next.js application.

## Key Accomplishments
- **Next.js 14 Integration**: Leverages the App Router and React Server Components.
- **Tailwind CSS Styling**: Modern, responsive UI with a focus on usability.
- **NextAuth.js**: Fully integrated authentication on the frontend, communicating with the FastAPI backend.
- **User Dashboard**: Initial layout with sidebar, profile summary, and match score visualization.

## Setup
```bash
cd frontend
npm install
npm run dev
```
